# 10/07/2017    V3.2.1.0
## 根据台湾客户提出的需求，增加如下功能：

* 输入用户号码后，通过关键字查询日志。

```
dhss:
    pgw:
      mmc: 466
      country-code: 886
      not-standard-country-code: '09'
      search-log-mode: true
```
配置search-log-mode 为 true，即可在页面显示关键字输入框。
